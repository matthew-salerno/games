My Games

These are some of my favorite projects from my ECE 4525 Course: Video Game Design and Engineering. All games are written purely with javascript and the p5 library. Each week we had to write a new game, so some of these games are more rushed than others. Most of my time was spent expanding my "engine", which implemented a simple component based system for entities to interact.

Bee Game (simple puzzle)

A simple puzzle game revolving around buttons. The positioning of characters in the instructions used to be correct in firefox, but it seems an update has changed that.

Tank Game

A tank battle game, of note is the new collision logic.

Pacman

A Pacman clone. Ghosts will hunt the player using A* if they are close enough.

3D Table

A 3d rendering of a table. This uses entirely 2d draw functions and linear algebra to achieve a single point perspective view.

Space Scout Pinball

The final project, based off an old windows favorite of mine, it mostly explores fun physics opposed to fun game design.
